1) формат команд:
	chmod, find

Общий формат команды:
command [-flag] [-flag params] [params],
	- command      - имя комманды;
	- -flag        - флаги, однобуквенные параметры запуска команды (режим работы, по-сути);
	- -flag params - параметры флагов , аргументы, относящиеся только к какомму-то конкретному флагу. пишутся сразу посл него
	- params       - объекты для обработки
	
Chmod:
	chmod - команда, позволяющая менять доступ к файлу.
	
	нужные библиотеки:
		<sys/stat.h>
		<fcntl.h>
	
	формат:
		int chmod(char* path, mode),
		где mode - новый режим открытия файла. Задается специальными константами, которые можно найти в man 2 chmod
		
Find:
	find - bash команда для поиска файлов в файловой системе.
